# Stutter / Repeated Filler Words

## Pattern

The same word or short phrase repeated 2-3 times consecutively:

```javascript
const stutterPatterns = [
  // English
  'and and', 'but but', 'so so', 'like like', 'the the',
  'and and and', 'so so so',
  // Chinese (kept for bilingual content)
  '那个那个', '就是就是', '然后然后', '这个这个', '所以所以'
];
```

Also detect the pattern dynamically: any single word repeated ≥2 times in a row.

## Deletion strategy

Delete the front repetitions, keep the last one.

```
Original: "and and we need to"
Delete:   "and"
Keep:     "and we need to"

Original: "like like I was saying"
Delete:   "like"
Keep:     "like I was saying"
```
