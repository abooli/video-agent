# Filler Word Detection

## Filler word list

```javascript
const fillerWords = [
  // Vocal fillers
  'um', 'uh', 'ah', 'er', 'hmm',
  // Discourse fillers (flag but don't auto-delete)
  'like', 'right', 'okay', 'so', 'actually', 'basically', 'literally',
  // Chinese fillers (kept for bilingual content)
  '嗯', '啊', '哎', '诶', '呃', '额', '唉', '哦', '噢', '呀', '欸'
];
```

## Deletion boundary

```
Wrong: delete filler's own timestamps (filler.start - filler.end)
       → may clip the tail of the preceding word

Correct: from previous word's end to next word's start
         → (prevWord.end - nextWord.start)
```

## User preference

- **Auto-delete**: `um`, `uh`, `ah`, `er`, `hmm` — pure hesitation sounds
- **Flag for review**: `like`, `right`, `so`, `actually`, `basically` — may be intentional
- Keep some `um`/`uh` where they serve as natural breath pauses between ideas
