# Core Principle

## Keep the later version

The later attempt is usually more complete:
- Delete the earlier one (slip / fragment)
- Keep the later one (complete sentence)

## Exceptions

If the user says "the first version was better", keep front, delete back.
If the user says "the second version was better", keep back, delete front (default).
